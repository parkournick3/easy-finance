# Get Started

Welcome to Easy Finance!

## Prerequisites

- Node.js
- pnpm
- Docker and Docker Compose or a PostgreSQL database

## How to run the project

1. Clone the repository
2. Run `pnpm install`
3. Run `cp .env.example .env`
4. Fill in the `.env` file with your own values ( or Run `pnpm db:start` to start a local database with docker and docker compose )
5. Run `pnpm db:migrate` to run the migrations
6. Run `pnpm dev`
7. Open [http://localhost:3000](http://localhost:3000) in your browser
